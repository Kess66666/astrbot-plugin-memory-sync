# 🕊️ AstrBot 跨实例记忆同步插件

> 通过 A2A 协议自动同步 `learnings/` 目录中的经验记录

## 快速开始

1. 安装插件到 `data/plugins/astrbot_plugin_memory_sync/`
2. 配置 `config.yaml` 中的远程 A2A 地址和 Token
3. 启动插件，后台自动同步！

## 功能特性

- ✅ 自动监控 `learnings/` 目录文件变更
- ✅ 基于时间戳智能去重
- ✅ 通过 A2A 协议发送到远程实例
- ✅ 可配置轮询间隔、目标地址、Token
- ✅ 提供 `/sync_test` 测试指令
